Keymaker CLI

Keymaker is a no-dependency Rust CLI tool that generates cryptographically secure random keys of any length in bytes. It works on Unix/macOS using /dev/urandom and on Windows via BCryptGenRandom.

Features

  • No external crates — pure Rust + OS APIs
  • Cryptographically strong randomness
  • Supports hex, base64, and raw output formats
  • Optional uppercase HEX output
  • Optionally save output to a file

Build Instructions

# Clone or place main.rs in a new Rust project
cargo build --release

Usage

keymaker <length> [--format hex|base64|raw] [--uppercase] [--out PATH]
  • <length> — Number of random bytes to generate (must be > 0)
  • --format — Output format (default: hex)
  • --uppercase — Use uppercase HEX (only valid with --format hex)
  • --out PATH — Write output to a file instead of stdout

Examples

# 32-byte key, hex (default)
keymaker 32

# 64-byte key, base64
keymaker 64 --format base64

# 16-byte raw key saved to a file
keymaker 16 --format raw --out secret.key

# 32-byte uppercase hex key
keymaker 32 --uppercase

Platform Notes

  • Unix/macOS: Uses /dev/urandom
  • Windows: Uses BCryptGenRandom (no additional libraries required)
